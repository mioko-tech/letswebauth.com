# Let's Webauth

[![License](https://img.shields.io/github/license/mioko-tech/letswebauth.com?color=blue&style=flat-square)](https://github.com/mioko-tech/letswebauth.com/blob/master/LICENSE)

This is the public repo for [https://letswebauth.com](https://letswebauth.com). This project is open-source so please raise issues and open PRs if you would like to see specific changes and improvements.

## Building
Make sure you have npm and gulp installed, then run:  
1) `npm install`  
2) `gulp dist` for one-off build of the dist directory, or `gulp watch` to build and watch for changes in `src`  
3) `open dist/index.html`
