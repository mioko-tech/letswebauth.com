About

This site is created by Mioko, creators of a Web Authentication product named Presence which is linked to in various locations. Presence is a paid-for product with an initial free trial period.

If you are interested in being involved, sponsorship, partnership, promotion and/or aquisition, please contact us at contact@mioko.tech

Roadmap

This site currently focuses on platform authentication without use of attestion services. In future we would also like to add details for security keys and attestation.

Note

Please note this site not currently affiliated in any way with the awesome organisation that is Let's Encrypt. The goal of Let's Webauth is to invoke a community spirit to learn, adopt and grow this new technology for the benefit of web applications around the world.
