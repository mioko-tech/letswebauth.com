Web Authentication (WebAuthn) is a new open technology that allows users to quickly, securely and easily log in to your website using biometrics such as fingerprint or face identification. Supported on all modern Android and iOS devices, we are encouraging it's use and are helping developers around the world to learn more about it!

What is Web Authentication? Show me the detail!

As of September 2020 it is realistic and acheivable to allow the vast majority of mobile web users to sign into your website using biometrics (fingerprint and face identification).

Key points:

  • Web Authentication is available on both Android and iOS, meaning it's available for up to 99% of mobile users[1] and widespread adoption. This website is particularly focused on the platform authentication application of Web Authentication to fit with this.
  • Biometrics always stay on the user's device
  • Encrypted keys are used to exchange information with the server.
